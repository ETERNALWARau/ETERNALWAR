# Create a Wallet

To get started on The Eternal War the first thing you'll need is to set up a wallet that supports Binance Smart Chain (BSC). Wallets are available both on desktop computers and on smartphone devices. You'll need to choose the wallet that fits your needs best.

{% hint style="danger" %}

* ‚úÖ **Download and install only the latest version from an official source.**

* ‚úÖ **Safely back up your recovery phrases.**
* ‚ùå **NEVER share your recovery phrases with anyone, under any circumstances.**
* ‚ùå **NEVER input your recovery phrase to a website or app, other than your wallet app.**
